这是什么？
-------
Anabyss是一款大型的、冒险的、科技的、生存的、跨维度的Minecraft模组。  
在这个模组里面，拓展了下界的下层维度的世界观，也就是这个模组的核心模组的世界观——深渊。  
有关深渊的更多信息，你可以在[这篇文档](http://175.27.232.224/anabyss.html)中找到。
